CNN网络发展史

发表于 2019-03-28 | 更新于 2019-03-28 | 分类于 深度学习 | 阅读次数
字数统计 139 | 阅读时长 1

卷积神经网络可谓是现在深度学习领域中大红大紫的网络框架，尤其在计算机视觉领域更是一枝独秀。 CNN从90年代的LeNet开始，21世纪初沉寂了10年，直到12年AlexNet开始又再焕发第二春，从ZF Net到VGG，GoogLeNet再到ResNet和最近的DenseNet，网络越来越深，架构越来越复杂，解决反向传播时梯度消失的方法也越来越巧妙。本文总结一波CNN的各种经典架构。
